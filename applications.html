Bank @ Home v1.2.3
 An online banking simulation created utilizing the Visual Basic 6.0 platform.

Users can create a savings, checking or a christmas club account with the ability to transfer, withdraw or deposit funds.

The data is stored in an Microsoft Access database.

 

CustomerWorks v1.0.0

A customer tracking application created utilizing the Visual Basic 6.0 platform.

This application was caliborated with two other developers.

Features include: view payment and service history, open and search accounts for customers for a ficticous cable company.

The data is stored in an Microsoft Access database.

   
DPL Federal Credit Union Inventory An inventory tracking application created utilizing the Visual Basic 6.0 platform.

Electronics or various equipment is tagged with a ID with the ability to search, add, edit or delete entries.

The data is stored in an Microsoft Access database.
Lock Bag Journal A journal application created utilizing the Visual Basic 6.0 platform.

When lockbags are logged, the user has the ability to keep entries of what information was stored and allows the ability to add, edit or delete entries.

The data is stored in an Microsoft Access database.

WinMP3 v1.0.1
 A media player application created utilizing using the Visual Basic 6.0 platform.

Users can listen to .mp3 files with the ability to create custom playlists, choose from five color themes while using minimal system resources.

*Not compatible with Microsoft Windows Vista.
